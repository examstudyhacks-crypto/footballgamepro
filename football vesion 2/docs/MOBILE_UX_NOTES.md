# Mobile UX Notes

This Streamlit app has been shaped to feel more like a mobile game than a desktop dashboard.

## Mobile-first decisions

- Sidebar starts collapsed so the game controls stay in the main screen.
- Main container has a narrower max width for easier reading.
- Hero, instructions, mode cards, fixture cards and winner graphic use responsive CSS.
- Large buttons reduce mis-taps on phones.
- Fixture cards appear before full tables so the next action is clear.
- Data-heavy areas are kept inside tabs after the primary simulate action.
- The Help tab repeats the rules inside the active game so users do not need to reset.

## Suggested future upgrades

- Add club badges or generated crests.
- Add a manager career mode.
- Add live transfer windows between seasons.
- Add a hall of fame page for previous winners.
- Add downloadable share cards for big finals.
