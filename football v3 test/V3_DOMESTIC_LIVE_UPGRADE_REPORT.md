# V3 Domestic + Live Gameplay Upgrade Report

Added:

- Premier League mode with 20 teams and a 38-matchday double round-robin.
- FA Cup mode with 64-team random knockout draw, extra time and penalties.

- Premier League and selected FA Cup player cores so scorers/assisters feel real.

- Mobile bottom action menu with Watch Live, Quick Sim and Show Results.
